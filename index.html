Guess ¿

Can you guess what the other person has written ?

Rules

  1. Both players have to type in there names into the players box.

  2. Then the questioner will set the numeric amount of attempts they will give there opponent.

  3. The questioner will then type in a question into the desginated box for questions.

  4. The opponent must try to answer the question in the amount of attempts the questioner has given them

  5. If the opponent does not answer in the x amount of attempts the questioner gets a point.

  6. First player to 3 wins.

  7. Good Luck!
